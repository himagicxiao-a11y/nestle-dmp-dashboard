📐 计费逻辑说明

🔁 Retargeting人群推送

总人天 = 需求确认×0.1 + 上传人群包数×0.02
费用 = 总人天 × 3,000元/人天

示例：100个上传包 → 0.1+100×0.02=2.1人天 → 费用6,300元

🔍 ID圈包/重合度计算

总人天 = 需求沟通0.1 + 圈包任务×0.01 + 报告整理0.3
费用 = 总人天 × 3,000元/人天

示例：10个圈包任务 → 0.5人天 → 费用1,500元

📊 人群画像数据

总人天 = 需求沟通0.1 + 圈包任务×0.01 + 报告整理0.5
费用 = 总人天 × 3,000元/人天

示例：10个画像任务 → 0.7人天 → 费用2,100元

🛒 电商效果评估（MET）

总人天 = 需求沟通0.5 + 包数×0.02 + 分析包数×1×跨平台+20%
费用 = 总人天 × 3,000元

示例：9个分析包×3平台 → 约15人天

🏗️ DMP平台搭建

218,200元/年，按12个月摊销
= 18,183元/月

示例：2个月消耗 → 18,183×2≈36,366元

🔄 ID转译（MAC/BDID→DID）

按实际转译成功量计费
单价：0.04元/条

示例：转译455万条 → 4,554,655×0.04≈182,186元
